Nome dell'Esercizio (Es. 1: Calcolo Fattoriale)

Inserisci qui le istruzioni dell'esercizio. Ad esempio: Scrivi un programma in C++ che chieda all'utente di inserire un numero intero non negativo e calcoli il fattoriale di quel numero. 


#include <iostream>

int main() {
    int n;
    long long fattoriale = 1;
    
    std::cout << "Inserisci un numero intero: ";
    std::cin >> n;

    if (n < 0) {
        std::cout << "Errore: il fattoriale di un numero negativo non esiste." << std::endl;
    } else {
        for(int i = 1; i <= n; ++i) {
            fattoriale *= i;
        }
        std::cout << "Fattoriale di " << n << " = " << fattoriale << std::endl;
    }

    return 0;
}
